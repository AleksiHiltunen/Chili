Functions

System

Exit Chili

Description

Command to quit the current Chili program

Command options

exit
quit
end

Parameters

-

Example(s)

()o_o) exit

Run Script

Description

Command to run premade scripts on an active robot. Script files are recommended to be .txt files.

Command options

script

Parameters

Complete path to the script file

Example(s)

()o_o) script C:\temp\script.txt

Autonomous Life

Description

Robots autonomous life is the setting when it moves as it speaks or idles. Autonomous life setting has a few levels of activity

Command options

autolife
auto_life
autonomous_life

Parameters

disabled / contextual / solitary

Example(s)

()o_o) autolife disabled

Wait

Description

Wait command to do nothing before starting to execute next command.

Command options

wait

Parameters

Wait time in seconds

Example(s)

()o_o) wait 5

Motion

Move

Description

Command to move a robot in x and y directions and degrees around its axis.

Command options

move
walk

Parameters

forward=meters
back=meters
left=meters
right=meters
clockwise=degrees
counterclockwise=degrees

Example(s)

()o_o) move forward=1
()o_o) walk left=0.4
()o_o) move back=2 clockwise=180

Dance

Description

Command to make robot dance a known dance.

Command options

dance

Parameters

Name of the dance
Installed dances: flossing, ymca, disco, wave

Example(s)

()o_o) dance disco

Animation

Description

Command to make robot perform a random animation of given animation tag. Animation tags are predenifed and each tag contain one or several different animations

Animation tags

Command options

animation

Parameters

Tag

Example(s)

()o_o) animation happy
()o_o) animation uncomfortable

Exploration

Description

Command to make a robot walk or wander around without a destination. Robot will stop moving if there are obstacles preventing its movement or limit is reached

Command options

wander
explore
move_around

Parameters

Radius limit in meters

Example(s)

()o_o) explore 5
()o_o) move_around 100

Joints

Description

Command to manipulate robots joints(left arm, right arm, head)

Command options

joint
rotate

Parameters

joint name=position
joints:
larm(left arm)
rarm(right arm)
head

positions:
up, side, front, side_elbow_bend

Example(s)

()o_o) rotate head=left
()o_o) joint larm=up

Command options

say
speak

Parameters

sentence

Example(s)

()o_o) say hello world!
()o_o) speak I am Pepper, a humanoid robot

Language

Description

Command to change current spoken language

Command options

language

Parameters

One of the following: Finnish / English / French

Example(s)

()o_o) language English

Volume

Description

Command to change spoken volume. Parameter is the percentage of the maximum volume

Command options

volume

Parameters

0-100

Example(s)

()o_o) volume 50
()o_o) volume 0

Camera

Take Picture

Description

Command to capture a photo using robots forehead camera. Picture is saved in robots memory. If picture identifier already exists in robots memory, it will be overwritten.

Command options

picture
shoot
photo
take_picture

Parameters

Picture identifier

Example(s)

()o_o) photo image.jpg
()o_o) picture new_photo.jpg
