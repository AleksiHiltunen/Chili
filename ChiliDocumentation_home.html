Chili

Chili is a third party software that uses SoftBank Robotic's NAOqi Python SDK in order to operate and code Pepper. As other robots of SoftBank Robotics use pretty much the same software as Pepper does, should Chili be compatible with them as well. Chili is part of Aleksi Hiltunen's Master's thesis, done in Tampere University of Technology.

Chili General

Chili can be used as a real-time interpreter. Real time interpreter takes in single commands, which are then executed on Pepper. This is also a good way of learning how commands are used. Real-time command line show the prompt "()o_o)" every time a new command is executable. If there is no prompt active, it means that previous command is still in run. Commands can be threaded (run in background) so that next command can be executed before the previous is done. This can be done by adding "&" character in the end of the command.

Example:
move forward=2 &
say Hey I'm moving forward and also speaking

With out the "&" option, Pepper would move forward 2 meters after which it would say the "hey I'm moving forward and also speaking"

Another way of using Chili is creating .txt files, that have line-by-line commands in it, that are then all executed in order. This way several commands can be executed repedetly without needing to type then again and again. Recursive script calling is also possible.

See example scripts in "Getting Started" section
