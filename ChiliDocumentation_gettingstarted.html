Example codes and scripts

My first script

  1. Open up a new text file
  2. Type in:
    language English
    volume 70
    say Hello, I am Pepper, it is my pleasure to meet you!
    animation bow
    animation hello
  3. Save the file, for example in C:\temp\script.txt
  4. Launch Chili
  5. Run the script with a command:
    script C:\temp\script.txt
  6. Check out Peppers actions
